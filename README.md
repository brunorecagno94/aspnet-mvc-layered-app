# ASP.NET MVC Layered Web Application

## Purpose
Academic project developed to apply Back-End and Full-Stack development concepts using the .NET ecosystem.

## Description
This project is a Full-Stack web application developed using ASP.NET MVC, following a layered architecture. It implements CRUD operations for core domain entities and integrates with a relational database through Entity Framework.
The application includes both the development and consumption of RESTful APIs for data exchange between application layers.

## Technologies
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- LINQ

## Architecture
- Layered architecture (Presentation, Business Logic, Data Access)
- Repository pattern
- Dependency Injection

## Features
- CRUD operations for domain entities
- RESTful API implementation and consumption
- Input validation and error handling
- Initial data seeding
- Separation of concerns for maintainability

## Concepts Applied
- Object-oriented programming (OOP)
- Clean code practices
- Layered architecture design
- API integration
- Data persistence with ORM
